# 🚗 WhatsNext Vision Motors – Salesforce CRM Project

## 📌 Project Overview

WhatsNext Vision Motors is a Salesforce CRM project developed to automate vehicle order management and improve customer experience.

The system automatically assigns the nearest dealer based on customer location, validates vehicle stock before order creation, updates order status automatically, and sends email reminders for scheduled test drives.

---

## ✨ Features

- Vehicle Management
- Dealer Management
- Customer Order Management
- Automatic Dealer Assignment
- Stock Validation using Apex Trigger
- Record Triggered Flow Automation
- Batch Apex for Stock Updates
- Scheduled Apex for Order Processing
- Email Notifications
- Reports and Dashboards

## 🔄 Project Workflow

1. Customer places a vehicle order.
2. System validates stock availability.
3. If stock is available:
   - Order is created.
   - Nearest dealer is assigned automatically.
4. If stock is unavailable:
   - Order creation is prevented.
5. Batch Apex updates stock availability.
6. Scheduled Apex updates order status.
7. Email reminders are sent for scheduled test drives.
